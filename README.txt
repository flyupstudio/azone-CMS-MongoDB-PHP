Установка
1) Распакуйте содержимое архива в папку или поддомен на веб сервере
2) Сервер при создании тестовой задачи был с такими параметрами (Apache2+nginx+php5.6+mongodb 3.2)
3) в файле /core/configuration.php В массиеве config надо указать имя базы данных где может ханиться колекции имейлов и урлов
